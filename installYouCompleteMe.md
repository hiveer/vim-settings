install YouCompleteMe in Vim based on 'vim-plug'

### clone

register in .vimrc like

`Plug 'Valloric/YouCompleteMe'`

then open Vim and run

`:PlugInstall`

### compile

`cd ~/.vim/bundle/YouCompleteMe/`

run `./install.py --all`

'--all' here means compile with everything enabled
we have other options you can check it on offical site: https://github.com/Valloric/YouCompleteMe#mac-os-x
